## 🌐 Overview

A dark-themed, highly interactive personal portfolio designed to bridge the gap between Conceptual AI and Production Software. Designed and developed from scratch, this web application showcases my software engineering projects, technical capabilities, and academic focus in Data Science and Full Stack Architecture.

**[View Live Project]({Insert_Your_Render_Link_Here})**

## ✨ Key Features

* **Dynamic Hero Dashboard:** A striking introductory layout featuring an "OPEN TO WORK" status indicator, immediate CV download options, and real-time API integrations displaying live LeetCode and GitHub performance metrics.
* **Interactive 3D Animations:** Engineered engaging UI/UX elements utilizing Framer Motion, including scroll-triggered section reveals, spring-loaded project cards, and a unique "Cyber-Construct" launch effect.
* **Component-Based Project Gallery:** A scalable "Selected Works" section managed via a centralized `projectData.js` file, cleanly showcasing full-stack system architectures like MediBot and NeonPulse with direct repository routing.
* **Technical Orbit Carousel:** A sci-fi inspired rotating interface displaying core competencies across frontend, backend, and cloud technologies.
* **Terminal-Inspired Academic Profile:** Structured biography cards detailing dual-degree pursuits at IIT Madras and LJIET, alongside technical focuses in Medical AI and Computer Vision.
* **Seamless Navigation:** A sleek, floating bottom navigation dock provides intuitive access to different sections.
* **Direct Transmission Portal:** An integrated contact form and communication hub enabling professional inquiries directly from the interface.

## 🛠️ Tech Stack

* **Frontend:** React.js, HTML5, JavaScript
* **Styling:** Tailwind CSS (Dark Mode Optimization)
* **Animation Engine:** Framer Motion
* **Deployment & Hosting:** Render
* **Data Management:** Local JSON/JS structuring (`projectData.js`)
